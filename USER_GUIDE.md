# 智能衣橱使用说明

欢迎使用**智能衣橱**！这是一款轻量级的智能衣物管理和搭配应用，帮助您轻松管理衣橱，获得每日穿搭灵感。

## 主要功能

### 📸 衣橱管理
在"衣橱"页面，您可以：
- **添加衣物照片**：点击右下角的"+"按钮，选择从相册选择或拍照
- **自动分类**：添加照片后，系统会提示您选择分类（大衣、外套、上衣、裤子、长裙、短裙、鞋子、配饰）
- **分类筛选**：使用顶部的分类标签快速筛选不同类型的衣物
- **查看和删除**：长按衣物卡片可以删除该衣物

### ✨ 智能搭配
在"搭配"页面，您可以：
- **自动生成搭配**：应用会根据您的衣橱自动生成穿搭建议
- **换一套**：不满意当前搭配？点击"换一套"按钮生成新的搭配
- **替换单品**：点击搭配中的任意单品，可以替换为同类别的其他衣物
- **点赞保存**：看到喜欢的搭配，点击"♥ 点赞保存"按钮收藏起来

### ❤️ 收藏管理
在"收藏"页面，您可以：
- **查看所有收藏**：浏览所有点赞保存的搭配
- **查看详情**：点击搭配卡片查看完整的搭配详情
- **删除收藏**：长按搭配卡片或在详情页面点击删除按钮

## 搭配规则

智能搭配算法会根据以下规则生成穿搭建议：
- **必选**：上衣 + 下装（裤子或裙子）
- **可选**：外套/大衣（50%概率）
- **可选**：配饰（鞋子、帽子、围巾等，70%概率，最多2件）

## 使用技巧

1. **建立完整衣橱**：添加更多衣物照片，应用能生成更丰富的搭配建议
2. **分类准确**：正确分类衣物有助于生成合理的搭配
3. **定期整理**：删除不再穿的衣物，保持衣橱整洁
4. **保存灵感**：看到喜欢的搭配及时保存，方便日后参考

## 数据存储

- **本地存储**：所有数据（照片和搭配记录）都存储在您的设备本地
- **隐私保护**：无需联网，您的衣橱数据完全私密
- **轻量设计**：照片会自动压缩，不会占用过多存储空间

## 最低要求

- **衣物数量**：至少需要1件上衣和1件下装才能生成搭配
- **权限需求**：
  - 相册访问权限（用于选择照片）
  - 相机权限（用于拍照）

## 常见问题

**Q: 为什么无法生成搭配？**  
A: 请确保衣橱中至少有1件上衣和1件下装。

**Q: 如何修改衣物的分类？**  
A: 目前需要删除后重新添加。未来版本将支持直接编辑分类。

**Q: 照片会占用很多空间吗？**  
A: 不会。应用会自动压缩照片并生成缩略图，大幅减少存储占用。

**Q: 数据会同步到云端吗？**  
A: 不会。所有数据都存储在本地，保护您的隐私。
