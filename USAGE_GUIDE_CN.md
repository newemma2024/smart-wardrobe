# Smart Wardrobe 使用指南（修改版）

## 修改内容概述

本应用改进了基于文件夹的图片导入功能。应用会自动创建专用文件夹，用户只需通过USB或文件传输将图片复制到对应文件夹即可批量导入。

## 新的使用方法

### 1. 启动应用
启动应用后，会自动创建以下文件夹结构：

```
{应用文档目录}/WardrobeImport/
├── 大衣/
├── 外套/
├── 上衣/
├── 裤子/
├── 长裙/
├── 短裙/
├── 鞋子/
└── 配饰/
```

### 2. 查看文件夹路径
1. 打开「导入」标签页
2. 在屏幕顶部可以看到文件夹路径
3. 点击「复制路径」按钮将路径复制到剪贴板

### 3. 复制图片

#### 方法A：通过USB从电脑复制
1. 用USB线将手机连接到电脑
2. 打开文件资源管理器（Windows）或访达（Mac）
3. 导航到复制的文件夹路径
4. 将衣物照片复制到对应的分类文件夹中

#### 方法B：使用文件管理应用
1. 打开手机的文件管理应用
2. 导航到复制的文件夹路径
3. 从其他文件夹复制衣物照片

### 4. 扫描图片
1. 返回应用的「导入」标签页
2. 点击「刷新统计」按钮
3. 会显示每个分类中的图片数量

### 5. 执行导入
1. （可选）勾选「导入后删除源文件」
   - 勾选后，导入完成会自动删除原始文件
2. 点击「开始导入」按钮
3. 等待导入完成
4. 显示完成消息

### 6. 查看衣物
1. 打开「衣橱」标签页
2. 可以看到已导入的衣物
3. 可以按分类筛选

## 支持的图片格式

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)

## 注意事项

### 存储空间
- 图片会自动压缩，但大量导入时请注意存储空间
- 使用「导入后删除源文件」选项可以节省存储空间

### 文件名
- 文件名可以随意设置
- 即使文件名相同也没问题（应用会自动分配唯一ID）

### 文件夹结构
- 请勿修改子文件夹的名称
- 否则应用将无法识别

### 重复导入
- 多次导入相同图片会重复注册
- 使用「导入后删除源文件」选项可以防止重复

## 故障排除

### 找不到图片
- 确认文件夹路径是否正确
- 确认图片文件扩展名是否为支持的格式
- 点击「刷新统计」重新扫描

### 导入失败
- 检查存储空间
- 确认图片文件是否损坏
- 重启应用后重试

### 找不到文件夹
- USB连接时选择「文件传输」模式
- 在文件管理应用中查找「内部存储」→「Android」→「data」→「应用名」→「files」

## 计划添加的功能

## 开发者信息

### 文件夹路径
iOS: `/var/mobile/Containers/Data/Application/{UUID}/Documents/WardrobeImport/`
Android: `/data/data/{package_name}/files/WardrobeImport/`

### 访问方法
- iOS: 通过iTunes文件共享或Finder访问
- Android: 通过USB连接，启用文件传输模式
