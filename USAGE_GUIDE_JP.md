# Smart Wardrobe 使用ガイド（修正版）

## 修正内容の概要

このアプリは、フォルダベースの画像インポート機能を改善しました。アプリが自動的に専用フォルダを作成し、ユーザーはUSBやファイル転送で画像をコピーするだけで一括インポートできます。

## 新しい使い方

### 1. アプリを起動
アプリを起動すると、自動的に以下のフォルダ構造が作成されます:

```
{アプリのドキュメントディレクトリ}/WardrobeImport/
├── 大衣/          (コート)
├── 外套/          (ジャケット)
├── 上衣/          (トップス)
├── 裤子/          (パンツ)
├── 长裙/          (ロングスカート)
├── 短裙/          (ショートスカート)
├── 鞋子/          (靴)
└── 配饰/          (アクセサリー)
```

### 2. フォルダパスを確認
1. 「导入」(インポート)タブを開く
2. 画面上部にフォルダパスが表示される
3. 「复制路径」ボタンでパスをクリップボードにコピー

### 3. 画像をコピー

#### 方法A: PCからUSB経由でコピー
1. スマートフォンをPCにUSB接続
2. ファイルエクスプローラー（Windows）またはFinder（Mac）を開く
3. コピーしたフォルダパスに移動
4. 衣類の写真を対応するカテゴリフォルダにコピー

#### 方法B: ファイル管理アプリを使用
1. スマートフォンのファイル管理アプリを開く
2. コピーしたフォルダパスに移動
3. 他のフォルダから衣類の写真をコピー

### 4. 画像をスキャン
1. アプリの「导入」タブに戻る
2. 「刷新統計」ボタンをタップ
3. 各カテゴリの画像数が表示される

### 5. インポート実行
1. （オプション）「导入后删除源文件」にチェックを入れる
   - チェックすると、インポート後に元ファイルが自動削除される
2. 「开始导入」ボタンをタップ
3. インポートが完了するまで待つ
4. 完了メッセージが表示される

### 6. 衣類を確認
1. 「衣橱」(ワードローブ)タブを開く
2. インポートされた衣類が表示される
3. カテゴリ別にフィルタリング可能

## 対応画像形式

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)

## 注意事項

### ストレージ容量
- 画像は自動的に圧縮されますが、大量の画像をインポートする場合はストレージ容量に注意してください
- 「导入后删除源文件」オプションを使用すると、インポート後に元ファイルが削除され、ストレージを節約できます

### ファイル名
- ファイル名は自由に設定できます
- 同じファイル名でも問題ありません（アプリが自動的にユニークなIDを割り当てます）

### フォルダ構造
- サブフォルダの名前は変更しないでください
- アプリが認識できなくなります

### 重複インポート
- 同じ画像を複数回インポートすると、重複して登録されます
- 「导入后删除源文件」オプションを使用すると、重複を防げます

## トラブルシューティング

### 画像が見つからない
- フォルダパスが正しいか確認
- 画像ファイルの拡張子が対応形式か確認
- 「刷新統計」ボタンで再スキャン

### インポートが失敗する
- ストレージ容量を確認
- 画像ファイルが破損していないか確認
- アプリを再起動して再試行

### フォルダが見つからない
- PCとUSB接続時に「ファイル転送」モードを選択
- ファイル管理アプリで「内部ストレージ」→「Android」→「data」→「アプリ名」→「files」を確認
